
# Simple LED Flasher Circuit

A basic **LED flasher circuit** designed and simulated for the **IEC Lab course**.
The circuit uses two **BC547 NPN transistors** in an **astable multivibrator** configuration to produce alternating LED blinking through RC timing.

## Components

* BC547 NPN Transistors ×2
* LEDs ×2
* Resistors: 220Ω ×2, 22kΩ ×2
* Capacitors: 22µF ×2
* 9V DC Supply

## Concepts Used

* RC time constant
* Capacitor charging&discharging
* Transistor switching
* Astable multivibrator

## Tools

* Multisim (Simulation)

## Project Type

IEC Lab Course – Academic Project
